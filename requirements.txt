Python 3.x: Ensure that you are using Python version 3.6 or higher. This version supports the necessary features and libraries used in the project.

NumPy:
    NumPy is a fundamental package for scientific computing in Python. It provides support for large, multi-dimensional arrays and matrices, along with a collection of mathematical functions to operate on these arrays. In this project, NumPy is used for data generation, statistical calculations, and handling numerical operations efficiently.

Matplotlib:
    Matplotlib is a plotting library for creating static, animated, and interactive visualizations in Python. It is used in this project to visualize the simulated data stream and highlight anomalies in real-time. The library allows customization of plots to enhance readability and aesthetics.
